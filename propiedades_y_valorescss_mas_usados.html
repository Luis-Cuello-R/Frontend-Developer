Layout (diseño): - display ➡ Determina cómo se muestra un elemento en la página. Puede ser "block", "inline", "inline-block", etc. Textos: - font-size ➡ Establece el tamaño del texto. - font-weight ➡ Define el grosor del texto (normal, negrita, etc.). - font-family ➡ Especifica la fuente utilizada para el texto. - text-align ➡ Alinea el texto dentro de su contenedor. - color ➡ Define el color del texto. Modelo de caja: - margin ➡ Espacio entre el borde del elemento y los elementos adyacentes. - padding ➡ Espacio entre el contenido del elemento y su borde. - border ➡ Borde alrededor del contenido y el padding. Tamaños: - width ➡ Ancho del elemento. - height ➡ Altura del elemento. Colores de fondo: - background ➡ Propiedad abreviada para establecer el color de fondo, la imagen de fondo y otros estilos relacionados con el fondo. - background-color ➡ Establece el color de fondo del elemento.
